# Agent Instructions

Build apps with the **Aura** framework. Skills in `.claude/skills/`:

- `/aura-operations` — define queries/mutations/actions
- `/aura-client` — React hooks, provider, SSR
- `/aura-data` — Prisma schema, pagination, reactive tracking
- `/aura-context` — ctx.* surface (auth, notify, storage, scheduler, agent)
- `/aura-realtime` — reactive invalidation (read/write keys, routing)
- `/aura-app-setup` — server wiring & deployment

Conventions & build: `.agents/rules/`
